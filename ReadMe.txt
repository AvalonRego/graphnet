POne Specific Features

- this version of graphnet has been modified to be able to handle only a hexagonal grid with 14 strings.

- the POne data readers/writers are more generic and can be used with different simulations.

- /graphnet/example/05_liquido contains files to convert *.h5 files to *.parquet files in the correct format for use in graphnet

- /graphnet/example/04_training contains various training scripts.
